Tipos de CSS

CSS Inline

style="" dentro de uma tag Html determina que este atributo só irá funcionar na tag selecionada

CSS Interno ou Incorporado

Declaração de CSS irá funcionar para todas as tags especificadas

Instruções de CSS são dadas dentro da página em que se deseja configurar. Utiliza-se a tag <style> </style> antes da tag </head>

CSS Externo

Será aplicado em todas as páginas que possuírem o seu link de referência

Hierarquia de CSS

Inline > Interno > Externo

Seletores e classes

Seletor

Indica qual elemento HTML que será trabalhado

body{
}

  • background:

p{
}

  • color:

Classes

Atributos especificados por um título. Toda tag que que tiver este título herdará a configuração. Nunca iniciar o nome das classes com números.

Deve-se inserir no documento CSS o título/nome da classe depois de um ponto; no documento Html este títlo deve ser inserido na tag que herdará as configurações.

Exemplo:
class="center" (na tag do documento Html)
.center{
} (no documento CSS)

ID

A tag terá um nome único. Modificações serão feitas apenas na tag que contenha o id. No documento CSS digita-se # e o título escolhido; no documento CSS insere-se na tag específica id="" (dentro das aspas insere-se o título). Nunca iniciar o nome do id com números. Não pode existir mais de um id com o mesmo título.

Coringa*

Todas as tags receberão as propriedades que forem especificadas dentro de:
*{
}

Agrupamento de seletores

Todas as tags que receberão as mesmas propriedaes podem ser agrupadas.

Exemplo:
h1, p{
color:red;
text-align:center;
}

Propriedades

Color: é a Cor texto.
Padrões: Hexadecimais, RGB, HSL ou nome.

background: background-color:cor de fundo
background-image: url("../indica o caminho da imagem") imagem de fundo
background-size:cover (a imagem preenche todo o espaço)
